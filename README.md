**NÜA Wellness Spa** es un sitio web premium y sofisticado diseñado exclusivamente para centros de estética, spas de lujo y clínicas de bienestar. Con una identidad visual refinada en **tonos dorados y negros**, el sitio ofrece una experiencia sensorial y elegante que refleja el lujo, la armonía y la exclusividad del centro.

---

## 🖤 Identidad Visual & Diseño
El diseño de NÜA Wellness Spa ha sido personalizado meticulosamente para ofrecer una experiencia estética premium:
*   **Paleta de Colores de Lujo:** Uso predominante de un negro profundo (`#050505`) como fondo, combinado con detalles y acentos en oro refinado (`#c5a880`), creando un contraste de alta gama, místico y sumamente profesional.
*   **Tipografía Elegante:** Combinación de fuentes modernas y de alta legibilidad como *Poppins* para el cuerpo y textos interactivos, y fuentes serif premium como *Cormorant Garamond* para títulos y acentos de lujo.
*   **Efectos Visuales Impecables:** Microanimaciones en botones, efectos de transición suave (fade-in), tarjetas de servicio con efectos de profundidad y elevación (glassmorphism sutil).

## 🛎️ Características Principales
El sitio web está estructurado en múltiples secciones optimizadas para brindar información clara y fomentar la reserva de servicios:

*   **`index.html` (Inicio):** Slider de bienvenida interactivo, introducción de la marca con contadores animados, catálogo visual de servicios y testimonios destacados de clientes.
*   **`about.html` (Nosotros):** Detalla la filosofía de bienestar, la historia, el compromiso del spa y estadísticas de satisfacción.
*   **`service.html` (Servicios):** Listado detallado de tratamientos exclusivos (Masajes Esenciales, Tratamientos Faciales Avanzados, Cuidado Corporal y Paquetes de Lujo).
*   **`price.html` (Precios):** Tabla comparativa de planes de suscripción mensual (Plan Básico, Plan Familiar, Plan VIP) con desglose de servicios incluidos.
*   **`appointment.html` (Citas):** Formulario interactivo de reserva en línea con selectores de fecha, hora y tratamiento seleccionado.
*   **`opening.html` (Horarios):** Información visual de horarios de atención al cliente en un diseño limpio y elegante.
*   **`team.html` (Especialistas):** Perfiles y redes de las terapeutas y asesoras de estética holística del spa.
*   **`testimonial.html` (Testimonios):** Carrusel interactivo con las reseñas y opiniones de clientes VIP.
*   **`contact.html` (Contacto):** Mapa de ubicación integrado, datos de contacto (teléfono, email) y formulario de mensajes con validación.

## 🛠️ Tecnologías y Librerías Utilizadas
El proyecto utiliza una combinación de tecnologías frontend robustas para garantizar compatibilidad, interactividad y velocidad:

1.  **Estructura y Maquetación:**
    *   [HTML5 Semantic Markup](https://developer.mozilla.org/es/docs/Web/HTML) — Para una estructura limpia y optimizada para SEO.
    *   [Bootstrap 4.4.1](https://getbootstrap.com/docs/4.4/) — Grid responsivo y componentes de interfaz premium.
2.  **Estilos y Preprocesadores:**
    *   [SCSS / Sass](https://sass-lang.com/) — Código modular, estructurado y reutilizable para la personalización de estilos.
    *   [CSS3](https://developer.mozilla.org/es/docs/Web/CSS) — Con variables personalizadas para el sistema de diseño (paleta dorada y negra).
3.  **Interactividad y Efectos:**
    *   [jQuery 3.4.1](https://jquery.com/) — Para la manipulación interactiva del DOM.
    *   [Animate.css](https://animate.style/) — Animaciones fluidas al cargar y navegar por el sitio.
    *   [Owl Carousel](https://owlcarousel2.github.io/OwlCarousel2/) — Sliders interactivos y responsivos para testimonios y planes.
    *   [Tempus Dominus (Moment.js)](https://tempusdominus.github.io/bootstrap-4/) — Selector estético de fecha y hora para el formulario de citas.
    *   [CounterUp & Waypoints](https://github.com/bfintal/Counter-Up) — Contadores numéricos que se activan dinámicamente al hacer scroll.

## 🚀 Cómo Ejecutar el Proyecto Localmente

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/davidm-afk/NUA-Wellness-Spa.git
    ```
2.  **Acceder al directorio:**
    ```bash
    cd "Spa NÜA"
    ```
3.  **Abrir en tu navegador:**
    *   Puedes usar extensiones como **Live Server** en VS Code.
    *   O simplemente abrir el archivo `index.html` en cualquier navegador web moderno.
